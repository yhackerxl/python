AccountHub

A program for creating and managing user accounts on computers.

Features

  • Create a user account
  • Create multiple accounts at once
  • Delete a user account
  • Delete multiple accounts at once
  • Show user accounts
  • Show account creation and deletion history

Installation

  1. Clone the repository: git clone https://github.com/yhackerxl/python/pycomplex/AccountHub.git
  2. Install the required packages: pip install -r requirements.txt

Usage

To run the program, execute the following command in a terminal:

python AccountHub.py
